# aws-cloudwatch-alarms
# AWS CloudWatch Monitoring and Alarms

## Project Overview
This project demonstrates monitoring EC2 instances using Amazon CloudWatch and configuring alarms with SNS email notifications.

## Services Used
- Amazon EC2
- Amazon CloudWatch
- Amazon SNS

## Monitoring Metrics
- CPU Utilization
- Status Check Failed
- Network Traffic
- Disk Metrics

## Steps Performed
1. Created an SNS topic.
2. Subscribed an email endpoint.
3. Confirmed the subscription.
4. Created CloudWatch alarms.
5. Configured alarm actions.
6. Tested alarm notifications.
7. Verified email alerts.

## Alarm Configuration
- Metric: CPU Utilization
- Threshold: 80%
- Period: 5 Minutes
- Notification: SNS Email

## Outcome
Successfully monitored EC2 resources and received automated email alerts.
